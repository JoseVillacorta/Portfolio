# Portafolio Personal

Este es mi repositorio de mi portafolio personal web. Este portafolio fue diseñado para mostrar mis proyectos, experiencia y habilidades técnicas como Desarrollador Backend, con un enfoque moderno y minimalista.

## Tecnologías Utilizadas

Este proyecto fue construido desde cero, enfocado en rendimiento y diseño UI/UX:

- **React.js** (Componentes Funcionales y Hooks)
- **Vite** (Build tool ultrarrápido)
- **CSS3 Vanilla** (Variables, CSS Grid, Flexbox, Glassmorphism)
- **Web3Forms** (Integración para envío directo de correos sin backend adicional)
